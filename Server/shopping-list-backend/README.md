# Dev

1. Clonar el proyecto
2. Copiar el ```env.tempplate``` y renombrar a ```.env```
2. Ejecutar
```
npm install
```
4. Levantar la imagen (Docker desktop)
```
docker-compose up -d
```
5. Levantar el backend de Nest
```
npm run start:dev
```
